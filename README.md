# An-lise-de-Comportamento-do-Cliente
Projeto de análise de dados que demonstra análise do comportamento do cliente, usando Python, SQL e Power BI

## Visão Geral

Este projeto demonstra um fluxo completo de análise de dados, desde o carregamento e tratamento das informações até a criação de dashboards e relatórios executivos para apoio à tomada de decisão.

O objetivo principal foi transformar dados brutos em insights estratégicos por meio de análise exploratória, consultas SQL, visualizações interativas e relatórios automatizados.

## O projeto contempla:

- Carregamento e preparação do conjunto de dados
- Limpeza e padronização das informações
- Análise Exploratória de Dados (EDA)
- Consultas analíticas utilizando PostgreSQL
- Construção de dashboards interativos no Power BI
- Geração automatizada de relatório em PDF
- Criação de apresentação executiva em PowerPoint
- Conjunto de Dados

O conjunto de dados utilizado contém informações relacionadas a indicadores operacionais e analíticos, permitindo análises descritivas e identificação de padrões relevantes para o negócio.

## Etapas do Projeto

# 1. Carregamento dos Dados

Importação dos arquivos e conexão com banco PostgreSQL para armazenamento e manipulação dos dados.

# 2. Limpeza e Tratamento

Processo de preparação dos dados para garantir consistência e qualidade analítica.

# 3. Análise Exploratória de Dados (EDA)

## Análises estatísticas e visuais para identificação de:

- Tendências
- Distribuições
- Correlações
- Outliers
- Padrões de comportamento

## 4. Consultas SQL no PostgreSQL

Desenvolvimento de consultas SQL para responder perguntas de negócio e gerar métricas estratégicas.

- Agregações
- Filtros
- JOINs
- CTEs
- Funções analíticas

## 5. Dashboard no Power BI

Criação de dashboards interativos para acompanhamento de KPIs e indicadores relevantes.

Principais recursos:

- Filtros dinâmicos
- Indicadores estratégicos
- Visualizações interativas
- Análises comparativas

## 6. Relatório em PDF

Geração automatizada de relatório contendo:

- Resumo executivo
- Principais insights
- Gráficos
- Indicadores analíticos

## 7. Apresentação em PowerPoint

Desenvolvimento de apresentação executiva com os principais resultados do projeto e recomendações estratégicas.

Resultados dos Dashboards

Os dashboards permitiram:

- Melhor visualização dos indicadores-chave
- Identificação de tendências e padrões
- Monitoramento de métricas estratégicas
- Apoio à tomada de decisão baseada em dados
- Automatização da análise de informações
